# 《赛博坦克：四季山海远征》产品规格

状态：已批准进入分阶段开发  
目标用户：6–12 岁儿童及家庭共玩  
产品形态：手机 / Pad / 桌面浏览器，离线优先，可扩展家庭云服务

## 1. 产品愿景

把当前波次射击游戏升级为“根据天气选择装备、驾驶坦克探索四季山海、完成修复与救援、让世界长期变好”的 3D 家庭冒险。

产品不以消灭数量作为唯一成就，而让孩子通过观察环境、选择零件、合作解决问题和展示创作获得参与感。

核心循环：

```text
查看天气与任务 → 选择路线 → 进入机械工坊改装 → 探索/战斗/救援
→ 带回星屑与蓝图 → 修复家园 → 解锁新的季节路线和创造工具
```

## 2. 设计支柱

### 2.1 世界会变化

季节必须改变道路、物理、声音、任务和可达区域，不能只更换颜色贴图。

### 2.2 改装能被感知

每个行走模块、炮弹和机体必须在 20 秒试驾内感受到差异。优先使用“更稳、更会转弯、能过水、能搭桥”等具体能力，避免只展示抽象百分比。

### 2.3 失败也有进展

每场任务至少保留探索星屑、图鉴发现或家园材料之一。儿童不会因为失败失去已经获得的零件。

### 2.4 合作大于竞争

双人模式提供驾驶、瞄准、修复、导航等互补职责；排行榜默认是本机家庭榜，不用陌生人排名制造压力。

### 2.5 内容可持续扩展

季节、地貌、天气、路面、零件、任务和生态发现均采用数据定义和策略插件，避免继续向主循环增加硬编码分支。

## 3. 世界结构

第一版不制作无缝超大开放世界，采用“世界地图 + 3–6 分钟区域关卡 + 可持续变化的家园”。每个区域由共享地貌骨架、季节状态、天气状态和任务模板组合。

### 3.1 六个山海区域

| 区域 | 标志地貌 | 关键玩法 | 区域守护者 |
|---|---|---|---|
| 云岭雪山 | 悬崖、索道、山洞、冰川 | 爬坡、雪崩预警、修复索道 | 山岳巨像 |
| 星河平原 | 河流、湿地、村庄、农田 | 修桥、护送、恢复供电 | 稻穗机械兽 |
| 水晶湖群 | 湖泊、岛屿、水下遗迹 | 冰面穿越、浮航、折射谜题 | 水晶鳐 |
| 蓝鲸海岸 | 沙滩、潮汐、灯塔、礁石 | 潮汐路线、救援、灯塔导航 | 机械蓝鲸 |
| 红叶峡谷 | 瀑布、古树、化石、岩桥 | 风场、落石、化石拼装 | 峡谷翼龙 |
| 四季家园 | 车库、苗圃、陈列馆、试车场 | 建造、收藏、拍照、共创 | 玩家共同成长 |

### 3.2 四季玩法矩阵

| 季节 | 视觉与声音 | 路面/物理 | 新路线 | 代表任务 | 推荐模块 |
|---|---|---|---|---|---|
| 春 | 融雪、花粉、溪流、蛙鸣 | 泥地降低抓地，河水上涨 | 浮木桥、浅水湾 | 播种、修桥、寻找迷路工蜂 | 越野履带、浮航模块 |
| 夏 | 强日照、雷雨、蝉鸣、海风 | 干地速度高，沙地与过热增加 | 干涸河床、海蚀洞 | 灯塔充能、雨水收集、海岛寻宝 | 沙地气囊轮、散热装置 |
| 秋 | 红叶、收获、强风、雁群 | 落叶降低转向，侧风推动轻机体 | 收割田、风车坡 | 运送果实、追风筝、修复风车 | 稳定宽轮、风能回收器 |
| 冬 | 积雪、极光、冰雾、风铃 | 冰面保留惯性，深雪减速 | 冻结湖面、冰洞 | 点亮雪村、破冰救援、极光观测 | 雪地钉轮、破冰能量核心 |

季节切换支持两种方式：

- 章节模式：由故事进度推动，适合首次体验。
- 自由探索：玩家在家园季节仪中选择，适合收集与复玩。

## 4. 天气与环境事件

天气需提前 8–12 秒提供图形预警，保证儿童能够理解并行动。

- 小雨 / 暴雨：降低普通轮抓地力，填充低洼水道。
- 雾：缩短视野，但声波雷达能标记目标。
- 雷暴：给电磁武器充能，也会短暂关闭部分装置。
- 强风：影响轻型坦克、叶片与漂浮物方向。
- 雪暴：降低视野，沿灯标或队友轨迹导航。
- 潮汐：周期性开放海滩洞穴或淹没低处通道。
- 极光：技能冷却加快，隐藏晶体显形。

## 5. 3D 机械工坊

机械工坊是第一阶段的核心交互场景，不做成普通列表商店。

### 5.1 操作

- 单指/鼠标拖动：旋转坦克。
- 双指/滚轮：缩放观察。
- 点击高亮部件：拆卸、吸附、安装。
- 横向拖动部件抽屉：浏览同类零件。
- 长按部件：显示“适合什么地形”和简短演示。
- 装备完成：机械臂、磁吸、火花、悬挂回弹动画。
- 一键试驾：进入 20 秒迷你试车场，不需要重新加载主关卡。

### 5.2 装备槽位

| 槽位 | 作用 | 首版数量 |
|---|---|---:|
| 机体 | 耐久、重量、技能和整体轮廓 | 3 |
| 行走模块 | 抓地、转向、越野和水面能力 | 5 |
| 炮塔 | 射击节奏与旋转速度 | 3 |
| 能量核心 | 炮弹功能与环境交互 | 6 |
| 工具模块 | 修复、扫描、牵引、搭桥 | 4 |
| 外观 | 配色、贴纸、车灯、尾迹、喇叭 | 24+ |
| 伙伴机器人 | 提示、收集、拍照与情绪反馈 | 3 |

### 5.3 行走模块

- 公路轮：速度高、转向直接，湿滑路面表现一般。
- 越野履带：泥地和碎石稳定，最高速度较低。
- 雪地钉轮：冰雪抓地强，普通路面转向略慢。
- 沙地气囊轮：沙滩、浅水稳定，车身弹性明显。
- 浮航模块：可进入深水，陆地加速较慢。
- 后续磁悬浮模块：跨越小裂缝，受电磁天气影响。

模块差异应同时反馈到轮胎旋转、悬挂高度、车身侧倾、声音、尘土/水花和操控曲线。

### 5.4 能量炮弹

- 星脉弹：稳定通用。
- 弹跳晶体：反射墙面，适合峡谷和晶洞。
- 冰霜胶囊：冻结浅水、降低目标移动。
- 泡泡修复弹：修复队友、桥梁和任务设施。
- 闪电链路：连接多个目标与导电机关。
- 种子核心：生成藤蔓桥、激活植物机关。
- 推力波：移动障碍和漂浮物，后续加入。

每次任务携带两种能量核心，战斗中通过大按钮或手柄肩键切换。切换时炮口颜色、准星、音效和弹道预览同步变化。

### 5.5 外观与表达

- 三个独立喷漆区域，不限制固定整车皮肤。
- 贴纸、号码、车灯、天线旗、尾迹、喇叭和胜利动作。
- 外观通过探索、图鉴和成就获得，不使用随机付费箱。
- 保存三套可命名方案，例如“雪山号”“海洋号”“我的最快坦克”。

## 6. 任务体系

任务由“地点 + 季节 + 天气 + 目标 + 变化结果”组合。

### 6.1 目标类型

- 修复：桥梁、风车、灯塔、索道、机器人。
- 救援：被困伙伴、动物机器人、机械蓝鲸。
- 护送：能源车、种子运输车、幼年守护者。
- 探索：洞穴、化石、极光点、沉船遗迹。
- 环境解谜：导电、结冰、推物、搭藤蔓桥。
- 守护：在限定时间保护设施免受失控机器干扰。
- 摄影：在正确天气与地点记录自然发现。

### 6.2 环境型 Boss

Boss 战由观察、躲避、修复和合作组成，而非单纯削减血条。

- 山岳巨像：避开落石，修复三处能量关节。
- 机械蓝鲸：清除缠绕物，引导它返回深海。
- 暴风云核心：使用避雷塔形成能量回路。
- 水晶鳐：利用反射弹点亮湖底晶柱。

## 7. 儿童与家庭体验

### 7.1 分龄辅助

| 能力 | 6–8 岁默认 | 9–12 岁默认 |
|---|---|---|
| 瞄准 | 自动吸附、自动发射可选 | 瞄准辅助、手动发射 |
| 改装 | 一键推荐、地形图标 | 参数比较、自由搭配 |
| 导航 | 光带、伙伴语音/文字提示 | 罗盘、地图标记 |
| 失败 | 原地重建、保留全部探索物 | 检查点重建、保留主要奖励 |
| 任务文字 | 一句话 + 动画演示 | 简短步骤 + 可选详情 |

### 7.2 成就与参与感

- 世界记得玩家：修复的桥、点亮的灯塔、种下的树长期保留。
- 四季探索图鉴：记录天气、动物机器人、地貌和稀有事件。
- 任务结束解释装备贡献，例如“雪地钉轮帮助你避免了 8 次打滑”。
- 家园陈列室展示坦克、照片、奖杯、化石和生态模型。
- 伙伴机器人用具体反馈鼓励观察、合作和尝试，不制造连续登录压力。

### 7.3 亲子协作

- 双坦克模式：两名玩家分别驾驶。
- 车组模式：P1 驾驶，P2 控制炮塔、扫描和修复，适合一台 Pad。
- 导航员模式：家长查看地图与天气，儿童驾驶执行。
- 无自由文本聊天；未来家庭分享默认仅房间码和预设表达。

## 8. 关键页面与流程

```text
基地首页
├─ 世界地图：区域、季节、天气、任务
├─ 机械工坊：机体 / 行走 / 炮弹 / 工具 / 外观
├─ 试车场：即时感受改装差异
├─ 家园：长期世界变化和陈列
├─ 图鉴：自然发现、照片、装备贡献
└─ 家长入口：时长、音量、联网、数据与辅助设置
```

首次体验控制在 90 秒：

1. 伙伴机器人展示即将下雪。
2. 儿童在工坊安装雪地钉轮。
3. 进入 20 秒试车场感受普通轮与钉轮差异。
4. 出发完成点亮雪村灯塔的小任务。
5. 回到家园看到灯塔模型永久亮起。

## 9. 商业与安全边界

- 核心任务、操控模块和炮弹能力不通过付费制造强弱差。
- 儿童档案默认本地保存，不要求注册。
- 若接入云相册、作品分享、排行榜或账号，必须增加家长同意、数据最小化、昵称过滤、数据删除和保留期限。
- 不使用随机付费箱、陌生人私聊、惩罚式连续登录和无限推送。

## 10. 成功指标

- 6–8 岁儿童在 90 秒内能完成第一次改装并说出它适合的地形。
- 80% 首次玩家能在没有成人解释的情况下进入任务。
- 改装后 20 秒内能感知至少一种操控差异。
- Pad 横屏主流设备 P95 帧时间低于 25ms；桌面保持 60fps。
- 每个季节至少提供一条新路线、一种新物理和两类专属任务。
- 连续 15 分钟体验中不出现必须阅读长段文字才能继续的节点。
